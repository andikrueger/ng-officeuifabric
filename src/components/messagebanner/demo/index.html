Message Banner Demo | <uif-message-banner>

In order for this demo to work you must first build the library in debug mode.

Basic usage

Show/hide

To render message banner, use the following markup:
<uif-message-banner uif-is-visible="vm.bannerToggle" uif-message="{{vm.message}}" uif-action-label="Testbutton" uif-action="customFunction()" uif-on-close="customCloseFunction()" />
